Kotlin interview tips
Use ArrayDeque, not Stack<T>
Stack<T> is legacy and synchronized. ArrayDeque gives you clean addLast/removeLast for stacks and addLast/removeFirst for queues.
getOrDefault vs getOrPut
Use getOrDefault when reading. Use getOrPut when you want insert-if-absent and get the value back — perfect for adjacency lists.
PriorityQueue is min-heap by default
Pass reverseOrder() for max-heap. Custom: compareBy { it.field } or Comparator { a, b -> ... }.
in operator for readability
x in set and x in map (checks keys) are idiomatic and readable. Prefer over .contains() in interviews.
IntArray vs Array<Int>
IntArray is primitive — use it for grids and frequency arrays. Array<Int> boxes each int. Matters for memory and speed.
Destructuring in loops
for ((k, v) in map) and forEachIndexed { i, v -> } keep code clean. Interviewers notice idiomatic Kotlin.
Array / IntArray O(1) access · fixed size
Create
val a = IntArray(5) // [0,0,0,0,0] val b = IntArray(5) { it } // [0,1,2,3,4] val c = intArrayOf(1, 2, 3)
Access & update
a[0] // read a[0] = 9 // write a.lastIndex // a.size - 1
Sort
a.sort() // in-place val s = a.sortedArray() // new copy a.sortDescending()
Two-pointer swap
var l = 0; var r = a.lastIndex while (l < r) { val tmp = a[l]; a[l] = a[r]; a[r] = tmp l++; r-- }
Prefix sum
val prefix = IntArray(n + 1) for (i in 0 until n) prefix[i+1] = prefix[i] + a[i] // range sum [l,r] = prefix[r+1] - prefix[l]
2-D grid
val grid = Array(rows) { IntArray(cols) } grid[r][c] = 1 // n×n identity: // Array(n) { i -> IntArray(n) { j -> if (i==j) 1 else 0 } }
Binary search
val idx = a.binarySearch(target) // returns index if found, negative if not // manual: var lo = 0; var hi = a.lastIndex while (lo <= hi) { val mid = lo + (hi - lo) / 2 when { a[mid] == target -> return mid a[mid] < target -> lo = mid + 1 else -> hi = mid - 1 } }

Stack (ArrayDeque) LIFO · prefer ArrayDeque over Stack<T>
Create
val stack = ArrayDeque<Int>()
Push / pop / peek
stack.addLast(x) // push stack.removeLast() // pop stack.last() // peek (no remove) stack.isEmpty()
Monotonic stack
val mono = ArrayDeque<Int>() for (x in nums) { while (mono.isNotEmpty() && mono.last() > x) mono.removeLast() mono.addLast(x) }
Next-greater / next-smaller problems
Valid parentheses
val stack = ArrayDeque<Char>() val pairs = mapOf(')' to '(', ']' to '[', '}' to '{') for (c in s) { if (c in pairs.values) stack.addLast(c) else if (stack.lastOrNull() != pairs[c]) return false else stack.removeLast() } return stack.isEmpty()
Queue (ArrayDeque) FIFO · BFS bread and butter
Create
val q = ArrayDeque<Int>()
Enqueue / dequeue
q.addLast(x) // enqueue q.removeFirst() // dequeue q.first() // peek front q.isEmpty()
BFS skeleton
q.addLast(start) val visited = mutableSetOf(start) while (q.isNotEmpty()) { val node = q.removeFirst() for (nei in graph[node]!!) { if (nei !in visited) { visited.add(nei) q.addLast(nei) } } }
Level-order BFS
var level = 0 q.addLast(start) while (q.isNotEmpty()) { repeat(q.size) { // snapshot size each level val node = q.removeFirst() for (nei in graph[node]!!) if (nei !in visited) { visited.add(nei); q.addLast(nei) } } level++ }
Tracks shortest path distance from source
PriorityQueue Min-heap by default · O(log n) add/poll
Min-heap
val minH = PriorityQueue<Int>() minH.offer(5); minH.offer(1) minH.peek() // 1 (min, no remove) minH.poll() // removes & returns min
Max-heap
val maxH = PriorityQueue<Int>(reverseOrder()) // or: PriorityQueue(compareByDescending { it })
Custom comparator
// Sort pairs by second value ascending val pq = PriorityQueue<IntArray>(compareBy { it[1] }) pq.offer(intArrayOf(1, 5)) // Multiple fields: PriorityQueue { a, b -> a[0] - b[0] }
K smallest / K largest
// K smallest → max-heap of size K val kSmall = PriorityQueue<Int>(reverseOrder()) for (n in nums) { kSmall.offer(n) if (kSmall.size > k) kSmall.poll() } // kSmall.peek() = k-th smallest
Dijkstra skeleton
val dist = IntArray(n) { Int.MAX_VALUE } dist[src] = 0 val pq = PriorityQueue<IntArray>(compareBy { it[0] }) pq.offer(intArrayOf(0, src)) // [dist, node] while (pq.isNotEmpty()) { val (d, u) = pq.poll() if (d > dist[u]) continue for ((v, w) in graph[u]!!) { if (dist[u] + w < dist[v]) { dist[v] = dist[u] + w pq.offer(intArrayOf(dist[v], v)) } } }
ArrayDeque (double-ended) O(1) at both ends · sliding window max
All ops
val dq = ArrayDeque<Int>() dq.addFirst(x); dq.addLast(x) dq.removeFirst(); dq.removeLast() dq.first(); dq.last() dq.peekFirst(); dq.peekLast()
Sliding window max
val dq = ArrayDeque<Int>() // stores indices for (i in nums.indices) { // drop indices outside window while (dq.isNotEmpty() && dq.first() < i - k + 1) dq.removeFirst() // drop smaller elements from back while (dq.isNotEmpty() && nums[dq.last()] < nums[i]) dq.removeLast() dq.addLast(i) if (i >= k - 1) result.add(nums[dq.first()]) }
O(n) classic — window of size k, get max per step

Trees & Graphs DFS / BFS traversal · Common structures
TreeNode Definition
class TreeNode(var `val`: Int) { var left: TreeNode? = null var right: TreeNode? = null }
DFS (Recursive)
fun dfs(node: TreeNode?) { if (node == null) return // Pre-order: process node here dfs(node.left) // In-order: process node here dfs(node.right) // Post-order: process node here }
DFS (Iterative)
val stack = ArrayDeque<TreeNode>() stack.addLast(root) while (stack.isNotEmpty()) { val node = stack.removeLast() // process node if (node.right != null) stack.addLast(node.right!!) if (node.left != null) stack.addLast(node.left!!) }
Graph Adjacency List
// Using Array of Lists (faster if n known) val adj = Array(n) { mutableListOf<Int>() } for ((u, v) in edges) { adj[u].add(v) adj[v].add(u) // if undirected }
Advanced Patterns DP · Backtracking · Union-Find
Union-Find (DSU)
class DSU(n: Int) { val parent = IntArray(n) { it } fun find(i: Int): Int { if (parent[i] == i) return i parent[i] = find(parent[i]) // path compression return parent[i] } fun union(i: Int, j: Int) { val rootI = find(i); val rootJ = find(j) if (rootI != rootJ) parent[rootI] = rootJ } }
Backtracking Skeleton
fun backtrack(start: Int, path: MutableList<Int>) { res.add(ArrayList(path)) // save copy for (i in start until nums.size) { path.add(nums[i]) // choose backtrack(i + 1, path) // explore path.removeLast() // un-choose } }
DP (Memoization)
val memo = HashMap<String, Int>() // or IntArray if 1D fun dp(i: Int, j: Int): Int { if (baseCase) return 0 val key = "$i,$j" if (key in memo) return memo[key]!! val res = minOf(dp(i+1, j), dp(i, j+1)) memo[key] = res return res }
